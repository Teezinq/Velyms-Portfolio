Project 1

Overview

This project explored how simplicity can be combined with strong layout and motion. The goal was to create a clear, minimal interface that feels effortless.

Process

Planning and execution involved sketches, modular code, and visual hierarchy testing. I focused on clarity and cognitive ease — reducing noise while keeping personality.

Reflection

This project taught me the value of restraint — that design communicates best when you remove everything unnecessary. The result is not emptiness but focus.
